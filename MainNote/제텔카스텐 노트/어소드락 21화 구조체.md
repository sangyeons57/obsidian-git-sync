#cpp #typedef #stuct
```Cpp
stuct _abc
{
	int a;
	int b;
	float c;
};
```
위가 구조체 struct이다 구조체는 배열처럼 자료형을 묶어서 사용하는거다

```Cpp
_abc t;
t.a = 10;
t.b = 11;
t.c = 11.2f;
```
이런 방식으로쓴다
하지만 Cpp에서는 이런 방식이 가능 하지만
C는 sturct 구조체를 사용할때 사용자정의 타입은 
```Cpp
struct _abc t;
```
이렇게 사용해야한다.

따라서 C에서는 사용자 타입을 제정의해서 사용하는데 
이게 바로 typedef 이다
```Cpp
typedef struct 구조체이름 제정의이름

typedef struct _abc
{
	int a;
	int b;
	int c;
}abc;

abc.a = 10;
abc.b = 11;
```

```Cpp
typedef INT int;

INT a = 10;
```
이렇게 int의 이름을 제정의할수도있다.
