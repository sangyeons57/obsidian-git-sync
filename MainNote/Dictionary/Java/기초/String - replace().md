### 내용

```Java
str.replace(a, b);
```
string str에 a값들을 전부 b 로 바꾼다


```Java
str.replaceAll(a,b);
```
정규식을 이용해서 a 를 b로 바꾼다

```Java
str.replaceFirst(a,b);
```
해당 문자열의 첫번째 a 값만 b 로 바꾼다
